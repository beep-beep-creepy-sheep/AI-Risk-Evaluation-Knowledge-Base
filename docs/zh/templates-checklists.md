# 模板与清单

## 安全使用预检查清单

- 示例是否全部为模拟数据或公开风格材料？
- 是否已移除私密标识符和保密事实？
- 提示词是否禁止生成投资建议？
- 提示词是否要求说明不确定性和假设？
- 专业使用前是否需要人工复核？

## 输出复核清单

- 事实是否由提供上下文或可核验公开来源支持？
- 计算和公式是否已独立检查？
- 输出是否避免确定性建议？
- 局限性是否清楚可见？
- 语气是否专业、克制、与证据匹配？
- 是否记录了提示词版本和复核意见？
