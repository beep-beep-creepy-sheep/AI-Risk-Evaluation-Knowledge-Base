# 导读

AI 可以帮助分析师更快地起草摘要、解释指标、检查逻辑和设计测试。但它也可能生成看似专业、实际并不可靠的内容。

这本手册提供一套实用的工作方法：如何设定边界，如何写提示词，如何识别风险，如何把失败转化为回归测试。内容以投资和组合分析风格的场景为背景，但全部使用模拟数据或公开风格示例。

## 基本原则

- 不在提示词和文件中放入敏感信息。
- 把 AI 输出当作草稿，而不是最终结论。
- 要求模型说明来源、假设、不确定性和验证方法。
- 用常规案例、边界案例和对抗性案例测试提示词。
- 记录提示词版本和修改原因，让改进过程可追溯。

## 阅读建议

先阅读安全使用原则，再进入提示词模式。遇到输出问题时，用风险分类定位问题，用评估设计章节补充测试，用模板与清单形成稳定的复核习惯。
