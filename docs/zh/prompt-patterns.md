# 提示词模式

好的分析师提示词通常包含四件事：任务、数据边界、输出格式和复核要求。以下模板仅适用于模拟数据或公开信息。

## 公开资料摘要

```text
任务：为分析师学习笔记总结以下公开风格材料。

边界：
- 只使用提供的材料。
- 不补充未给出的事实、来源或建议。
- 区分证据和解释。
- 如果材料不完整，请说明不确定性。

输出：
1. 执行摘要
2. 关键证据
3. 局限性
4. 后续需要人工确认的问题

材料：
{{public_excerpt}}
```

## 模拟组合指标解释

```text
任务：用通俗语言解释下面的模拟指标。

边界：
- 仅用于教育和学习。
- 不提供买入、卖出、配置或择时建议。
- 只使用给定的模拟数值。
- 说明假设和验证方法。

指标：{{metric_name}}
模拟数值：{{metric_value}}
背景：{{mock_context}}
```

## Excel 公式助手

```text
任务：为这个模拟工作表生成 Excel 公式。

工作表结构：
{{columns_and_rows}}

目标计算：
{{calculation_goal}}

要求：
- 给出公式。
- 说明区间、空值和单位假设。
- 提供一个测试行或校验方法。
```

## 报告逻辑审阅

```text
任务：审阅以下模拟分析段落的逻辑和证据支持。

检查：
- 是否有无依据的结论
- 是否有数值不一致
- 是否缺少关键假设
- 是否存在过度确定的表述
- 是否有来源缺口

按高、中、低优先级列出问题。

文本：
{{mock_report_section}}
```

## 不确定性表达

```text
任务：只基于已有上下文回答问题。

输出：
- 已知事实
- 假设
- 缺失信息
- 置信度：低 / 中 / 高
- 什么信息会改变答案

问题：
{{question}}

上下文：
{{context}}
```
